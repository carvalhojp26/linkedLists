\doxysubsection{Descrição detalhada}
Ponto de entrada principal para a aplicação de manipulação de matrizes. 

Este arquivo contém a função {\ttfamily main}, que serve como o ponto de entrada para uma aplicação que manipula uma matriz armazenada em um arquivo de texto (\char`\"{}data.\+txt\char`\"{}). A matriz é representada internamente como uma lista ligada bidimensional. O programa oferece ao usuário um menu de opções para visualizar a matriz, alterar elementos, adicionar ou remover linhas/colunas, e calcular a soma máxima sob certas condições.

As operações disponíveis são\+:
\begin{DoxyItemize}
\item Visualizar a matriz atual.
\item Atualizar um valor específico dentro da matriz.
\item Adicionar uma nova linha com valores especificados pelo usuário.
\item Adicionar uma nova coluna com valores especificados pelo usuário.
\item Remover uma linha selecionada pelo usuário.
\item Remover uma coluna selecionada pelo usuário.
\item Calcular e exibir a soma máxima de uma seleção de valores da matriz.
\end{DoxyItemize}

O loop principal do programa apresenta essas opções em um menu, executa a escolha do usuário e repete até que a opção de sair seja selecionada. Após a execução de cada operação, o usuário é solicitado a pressionar Enter para continuar, permitindo a visualização dos resultados ou mensagens de status.
